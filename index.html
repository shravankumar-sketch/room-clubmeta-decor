Assumptions from spec

  • Wallet accrues indefinitely, no cap.
  • Leaderboard — one per room, summed across all 6 objects, resets on completion.
  • Unlock order — all objects open in parallel, no fixed sequence.
  • Tier thresholds — tunable config (left), not fixed values.
  • Entry — Room icon sits directly in the Club screen's action row, no intermediate club-detail screen.
